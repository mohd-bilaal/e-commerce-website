# ZentraMart

ZentraMart is a simple static e-commerce college project with a premium Indian marketplace UI.

## Run Locally

Open `index.html` directly in your browser, or run a local server:

```powershell
py -m http.server 5500 --bind 127.0.0.1
```

Then visit:

```text
http://127.0.0.1:5500/
```

## Files

- `index.html` - website structure
- `styles.css` - responsive premium UI styling
- `app.js` - products, search, cart, wishlist, checkout, modals, dark mode
- `server.ps1` - optional PowerShell static server fallback

## Features

- Landing page with premium hero and deal sections
- Product grid with search, filters, ratings, discounts, and product details
- Cart, wishlist, checkout UI, coupon and price breakdown
- Login/signup UI
- Account page and seller/admin dashboard
- Dark mode, mobile bottom navigation, reward popups, and responsive design
